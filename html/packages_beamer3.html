Columns and blocks
There is a way to insert columns in a presentation like the example here written:
CODE
\begin{frame}
\begin{columns}
\begin{column}{0.3\textwidth}
some text\\
on\\
left
\end{column}
\begin{column}{0.3\textwidth}
\centering
some\\
text\\
right
\end{column}
\end{columns}
\end{frame}
An image can be put beside some text. The text can also be split and will be displayed above and under eachother. To have the image displayed you must put it in the same folder or even in a different folder but the correct path. Otherwise you will not have the image inserted. Here there is an example with the code that you can use:


CODE
\begin{frame}{Example of text next to image}
\begin{columns}[T]
\begin{column}[T]{5cm}
Some text above \\ some text under
\end{column}
\begin{column}[T]{2cm}
\includegraphics[height=3cm]{imagepath.png}
\end{column}
\end{columns}
\end{frame}






Blocks
A block can be easily put in the LaTeX code and can be useful to distinguish between what you want to write about.
CODE
\begin{frame}

\begin{block}{Block info title}
Information text
\end{block}

\begin{alertblock}{Alert block title}
Alert Text
\end{alertblock}

\begin{exampleblock}{Example block title}
Example text
\end{exampleblock}

\begin{definition}{Definition block title}
Definition text
\end{definition}

\end{frame}
