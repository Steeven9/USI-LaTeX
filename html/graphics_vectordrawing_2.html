Inserting TikZ diagrams in a table
The biggest problem here is to make the diagram fit inside the table. To do so,


CODE
\documentclass{article}
\begin{document}
\begin{table}
\begin{tabular}{lcr}
table & long table & not so long table \\
1 & 123456789000 & 3.14 \\
\end{tabular}
\end{table}
\end{document}


Inserting an image in a TikZ diagram
If you try including an image inside a TikZ diagram, you're probably going to meet the following problem. The program does not recognize the inputs as commands and confuses them for items. So what you get is a mess.




CODE
\documentclass{article}
\begin{document}
\begin{table}
\begin{tabular}{lcr}
table & long table & not so long table \\
1 & 123456789000 & 3.14 \\
\end{tabular}
\end{table}
\end{document}


The first thing I tried was using pgfplots package. In the result the image was moved on the right and it was rather small. On an attempt to scale it, I discovered that the image is bounded to the Item, so it is not possible to scale it independently from the diagram. So I tried using the package \usepackage{overpic}.


CODE
\begin{overpic}
[width=\textwidth]{Leslie_Lamport.png}
\end{overpic}
