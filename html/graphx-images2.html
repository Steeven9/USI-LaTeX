HOW TO PLACE IMAGES
After adding the image, it's time to choose where to place it and how. Initially, you will notice that the figure does not necessarily appear at the exact point where you enter the code in the .tex file. In fact, Latex could put images on the next page or where there is enough space. This happens because you may have a lot of text in the document. The \includegraphics command is undoubtedly the most important command of the graphicx package and is the one that allows you to insert an image into the document. The general syntax of the command \includegraphics is: /includegraphics [options] {filename.jpg} in which "options" indicates a list of one or more options separated by commas.The main options that can be used are: width (for width) and hight (for height). In the section on how to resize and rotate an image in more detail detail of the includegraphics command.
HOW TO SET THE FLOAT
s
To place the image in the right position it is necessary to set the float value for the figure environment. To do this you need to set the float by adding [h!] after the command "\begin{figure}". The possible values are:
  • h (here) - same position
  • t (up) - top of the page
  • b (down) - at the bottom of the page
  • p (page) - on an additional page
  • ! (substitute): force the specified position
