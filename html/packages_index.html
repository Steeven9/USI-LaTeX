PACKAGES INTRODUCTION
This section will help you to have a deeper understanding on packages. Latex is full of functionality already installed that you can use immediately. You can expand Latex functionality by adding extra packages inside your document. In fact a package is a file or collection of files containing extra LaTeX commands and programming which add new styling features or modify those already existing, without having to include tons and tons of code in the preamble of a document. Instead, we just add the line \usepackage{package name} and we have access to all the commands, symbols, environments, etc., that are in the package. We have pointed out the most important packages that you should know from document geometry manipulation to multi-language, or simply to extend supported colors.
