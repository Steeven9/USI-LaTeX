Comparing two columns of each document class



Article/Book/Report/Letter - Without twocolumn package

CODE


\documentclass {article} {book} {report} {letter}
\usepackage {xcolor}
\usepackage {geometry}
\usepackage [most] {tcolorbox}
\geometry {margin=0pt}
\newtcolorbox {bgbox} [1] [] {nobeforeafter, sharp corners, #1}
\begin {document}
\noindent
\begin{bgbox} [height=\paperheight,
colback=black,width=0.35\textwidth,rightrule=0pt]
\end {bgbox}%
\begin{bgbox}
[height=\paperheight, colback=white, width=0.65\textwidth]
\section* {Canto 1 - Incipit - Inferno}
Nel mezzo del cammin di nostra vita\\
ché la diritta via era smarrita.\\
Ahi quanto a dir qual era è cosa dura\\
esta selva selvaggia e aspra e forte\\
che nel pensier rinova la paura!\\
Tant’è amara che poco è più morte;\\
ma per trattar del ben ch’i’ vi trovai,\\
dirò de l’altre cose ch’i’ v’ho scorte.\\
Io non so ben ridir com’i’ v’intrai,\\
tant’era pien di sonno a quel punto\\
che la verace via abbandonai.\\
....
\end {bgbox}
\end {document}

This is an universal method for making a two column document, without using the twocolumn {option}. For this reason it works also for the letter class, which doesn't support twocolumn option.

Article/Book/Report - With twocolumn package

In this way we can easily represent an universal output for each class.
CODE


\documentclass [twocolumn] {article} {book} {report}
\begin {document}
\section* {Canto 1 - Incipit - Inferno}
Nel mezzo del cammin di nostra vita\\
ché la diritta via era smarrita.\\
Ahi quanto a dir qual era è cosa dura\\
esta selva selvaggia e aspra e forte\\
Tant’è amara che poco è più morte;\\
ma per trattar del ben ch’i’ vi trovai,\\
dirò de l’altre cose ch’i’ v’ho scorte.\\
Io non so ben ridir com’i’ v’intrai,\\
tant’era pien di sonno a quel punto\\
che la verace via abbandonai.\\
....
\end{document}
