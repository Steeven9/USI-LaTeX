  • 10pt, 11pt, 12pt
    The default is 10pt, which means that this is the value assumed if no size option is specified. All other font size declarations are relative to this standard size, so that the section titles, footnotes, and so on will all change size automatically if a different basic font size is selected.


    • They define the dimensions of the sheet.
  • letterpaper (11 × 8.5 in)
    It corresponds to the default setting for the article class, in the US format.
  • legalpaper (14 × 8.5 in) (21.6 x 35.6 cm)
    US Standard legal paper size.
  • executivepaper (10.5 × 7.25 in) (36.7 x 18.4 cm)
    US Standard executive paper size.
  • a4paper (29.7 × 21 cm) (11.7 x 8.3 in)
    The A series paper sizes are defined in ISO 216.
  • a5paper (21 × 14.8 cm) (8.3 x 5.8 in)
    The A series paper sizes are defined in ISO 216.
  • b5paper (25 × 17.6 cm) (9.8 x 6.9 in)
    The B series paper sizes are defined in ISO 216.


  • one column
    Default option for article, report and letter classes.
  • two column
    The twocolumn option allows you to compose the entire document on two columns. It is the default of Book class.
  • oneside
    Specifies a document that will consist of a single facade. It is a default oprion for article and report classes.
  • twoside
    Specifies a document that will consist of a double facade. It is a default oprion for book class.


  • titlepage
    It is a default option for book and report classes. It specifies a new page after the document title
  • notitlepage
    It is a default option for article class. It specifies no title page

  • openright
    The openright option, predefined in the book class, always starts a chapter on a right page. This option is not available in the article class
  • openany
    The openany option, predefined in the report class, starts a chapter in the next page available. This option is not available in the article class, because \chapter command is not defined in that class.
