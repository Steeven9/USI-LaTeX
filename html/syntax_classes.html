Document Class

Documents written with LaTeX can be interpreted in different classes, which are standard formats for some types of documents. The first command determines the global processing format for the entire document. Its syntax is:

\documentclass [option] {class}
\documentclass [option1, option2, etc.] {class}

The \documentclass command takes an argument (class), which describe the type of document Where some value of class must be given, while [options] may be omitted if default values are acceptable.
The most common classes are:
  • Article , for articles (usually scientific), short reports, documents, ...
  • Book , for book.
  • Report , for longer reports, thesis, small book, ...
There are many other non-standard classes for the most diverse documents: there are memos (which allows a lot of freedom in customizing the document), topsis and suftesi (for thesis and doctoral thesis) and beamer for presentations.

Article

Article is the most common class as well as the most flexible.


\documentclass{article}
\title{Hello World}
\begin{document}
\maketitle
Hello world, content
\end{document}

Book

Book is the most common class as well as the most flexible.


\documentclass{book}
\title{Hello World}
\begin{document}
\maketitle
\end{document}

Report

Report is the most common class as well as the most flexible.


\documentclass{report}
\title{Hello World}
\date{November 9, 2018}
\begin{document}
\maketitle
\end{document}
